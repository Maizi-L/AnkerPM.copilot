# AI产品经理工作台 Demo

基于飞书AI的多维决策支持系统 - 前端Demo原型

## 📋 项目简介

这是一个**AI驱动的产品决策支持系统**，帮助硬件产品经理通过五维评分模型进行新品规划决策。

### 核心功能
- ✅ **五维评分展示**：市场吸引力、用户需求强度、产品可行性、商业价值、风险等级
- ✅ **综合评分计算**：基于量化模型自动计算综合评分
- ✅ **人机协作机制**：PM可以通过评论、权重调整、场景选择来修正AI评分
- ✅ **产品方案对比**：自动生成3套产品方案（高端/大众/性价比）
- ✅ **决策建议输出**：根据评分给出明确的决策建议

## 🚀 快速开始

### 本地预览

1. 直接双击打开 `index.html` 文件
2. 或者在浏览器地址栏输入文件路径

### 使用Web服务器（推荐）

```bash
# 如果安装了Python
cd ai-pm-dashboard
python -m http.server 8000

# 然后访问 http://localhost:8000
```

## 🎨 设计说明

### 视觉风格
- **主色调**：Anker橙色 (#FF6B35)
- **辅助色**：深蓝灰 (#2C3E50)
- **背景色**：浅灰 (#F5F7FA)
- **圆角**：8-16px
- **阴影**：柔和投影

### 交互功能
1. **决策场景切换**：点击保守型/平衡型/激进型按钮，自动调整权重
2. **权重滑块**：拖动滑块实时显示当前权重值
3. **重新计算**：点击按钮根据PM的输入重新计算评分
4. **响应式布局**：支持桌面和移动端

## 🔄 导入飞书妙搭

### 方法一：ZIP压缩包导入

1. **打包项目**
   ```bash
   # 将 ai-pm-dashboard 文件夹压缩成 ZIP
   # 确保包含 index.html, css/, js/ 目录
   ```

2. **导入妙搭**
   - 打开飞书妙搭：https://miaoda.feishu.cn/home
   - 点击「新应用」→「空白应用」→「专家模式」
   - 点击「快速开发」
   - 上传 `ai-pm-dashboard.zip`

3. **初始化项目**
   在妙搭AI对话框输入：
   ```
   解压这个压缩包，覆盖当前应用的源码。
   这是一个纯前端静态页面，不需要后端和数据库。
   请确保所有文件正确加载，页面可以正常显示。
   ```

4. **连接多维表格**（可选）
   - 在妙搭中添加数据源
   - 关联已创建的飞书多维表格
   - 修改 `js/main.js` 中的数据读取逻辑

### 方法二：手动重建

参考 `index.html` 的结构，在妙搭中手动搭建：
1. 使用妙搭AI对话式生成各个区域
2. 调整样式和布局
3. 连接多维表格数据源
