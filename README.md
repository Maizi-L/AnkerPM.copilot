🌐 **在线访问：** [国际线路](https://anker-pm-copilot.vercel.app/) | [国内线路](https://ankerpm-copilot.pages.dev/)

## 📖 项目简介

AnkerPM.copilot 是一个 AI 原生产品规划工作台，旨在帮助硬件产品经理更高效地完成新品规划决策。

串联「信息采集 → AI 分析 → PM 共创 → 方案生成」全流程，让 AI 负责规模化采集和分析，PM 专注于价值判断和最终决策。

---

## 💡 主要功能

### 📱 持续监测用户真实声音
**痛点：** 用户/潜在用户需求信息碎片化，无法及时洞察潜在商机。

**解决方案：** 抓社媒 / 电商 / 调研零散反馈，自动聚类主题、量化情绪、识别潜在需求。

### 📊 动态捕捉行业增量信号
**痛点：** 市场变动频繁，风口与泡沫齐飞。

**解决方案：** 监控投融资资金流向、竞品动作、行业报告，定位"有需求、没产品"的空白带。

### 🎯 五维评分体系
**痛点：** 新品策划决策因素复杂，找"最优解"需多方斡旋。

**解决方案：** 综合计算新品 Idea 的市场吸引力、用户需求、产品可行性、商业价值、风险等级。

### 👤 PM 全程有最高权重
**痛点：** 纯 AI 工作流缺少人类产品经理的经验判断，方案可行性差。

**解决方案：** PM 可在任意环节介入，提交"直觉/经验"参数，随时为当前的方案纠偏。

## 🏗️ 系统架构

### 双环架构设计

本系统采用**外环（机会发现）+ 内环（人机共创）**的双环架构：

**外环 - 机会发现引擎（Always-On）**
- AI 持续监控市场，主动发现机会
- 无需 PM 介入，自动生成机会排行榜
- PM 从榜单中选择感兴趣的机会进入内环

**内环 - 人机共创定义流（Human-in-the-Loop）**
- PM 在关键决策点签字
- AI 负责执行和分析
- 人机协作完成产品定义

### 完整工作流程

```
用户进入工作台
  ↓
📊 机会发现（外环）
  └─ AI 持续监控 → 机会排行榜 → PM 选择机会
  ↓
📋 新建项目
  └─ 渐进式对话引导 → 创建项目
  ↓
📋 项目概览
  └─ 项目基础信息 + 工作流进度导航
  ↓
🛰️ 信息看板（内环开始）
  └─ 数据采集进度 → AI 初步分析 → PM 评论 → 融合分析
  ↓
📊 深度情报
  └─ VOC 分析 → 技术资产盘点 → 竞品功能矩阵
  ↓
🤝 人机共创
  └─ 需求澄清 → 红队压测 → 用户替身访谈 → 专家评审
  ↓
🎯 人机协作量化决策
  └─ 五维评分 → PM 调整权重 → 融合分析 → 综合决策
  ↓
📐 方案对比
  └─ 规划 → 三套方案生成 → 对比分析 → 文档输出
```

## 🔧 技术实现

### 飞书工具链集成

| 功能模块 | 飞书工具 | 集成方式 | 主要用途 |
|---------|---------|---------|---------|
| 信息采集与管理 | 多维表格 + 自定义应用 | API 集成 | 存储和管理各类信息源数据 |
| AI 分析与评分 | 飞书智能伙伴 (aily) + 自定义应用 | LLM API 调用 | 执行 NLP 分析、评分计算 |
| 产品经理交互 | 飞书自定义应用 + 多维表格应用模式 | 前端开发 | 提供交互式仪表板 |
| 方案生成 | 飞书文档 + 自定义应用 | 文档 API + 应用逻辑 | 自动生成方案文档 |
| 团队协作 | 飞书文档、多维表格、即时通讯 | 原生集成 | 支持团队评审和反馈 |
| 工作流管理 | 飞书自动化工作流 | 工作流编排 | 自动化流程执行 |

### 核心特点

- **🔄 双环架构**：外环持续发现机会，内环人机协作定义产品
- **🤝 人机协作**：AI 负责规模化采集和分析，PM 负责价值判断和决策
- **📊 量化决策**：五维评分体系，每个维度都有具体量化指标
