# AI产品经理工作台

一个基于AI的产品决策支持系统，帮助产品经理做出更科学、更高效的决策。

## ✨ 功能特点

- **信息协同**：自动采集多源数据（用户反馈、行业报告、竞品信息）
- **AI分析**：NLP情感分析、主题聚类、关键词提取
- **人机协作决策**：AI客观评分 + PM专业判断，五维评分体系
- **文档输出**：一键生成专业决策报告（Word/PDF/Excel/Markdown）

## 🚀 快速开始

1. 克隆项目
```bash
git clone https://github.com/Maizi-L/ai-pm-dashboard.git
```

2. 打开项目
```bash
cd ai-pm-dashboard
open index.html
```
