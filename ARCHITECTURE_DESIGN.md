## 1. 项目概述

### 1.1 产品定位
AnkerPM.copilot 是一个基于飞书生态的多 Agent 消费电子新品规划系统，串联「信息采集 → AI 分析 → PM 共创 → 方案生成」全流程。

### 1.2 核心价值
- **外环（机会发现引擎）**: AI 持续监控市场，主动发现机会
- **内环（人机共创定义流）**: PM 在关键决策点签字，AI 负责执行

### 1.3 目标用户
硬件产品经理

## 3. 数据流转图

```
用户进入工作台
  ↓
📊 机会发现页面（外环-全局）
  └─ AI持续监控市场，展示全品类机会排行榜
  └─ [选择机会并创建项目]
  ↓
新建项目页面
  └─ Step 1: 自由描述 + 上传文件
  └─ Step 2: AI引导式问答（5个问题）
  └─ 输出：项目基础信息
  ↓
📋 项目概览
  └─ 展示项目基础信息
  └─ 工作流进度导航
  ↓
🛰️ 信息看板（内环-针对当前项目）
  └─ 数据采集进度
  └─ 各渠道AI初步分析 + PM评论
  └─ 机会发现板块（基于当前项目数据）
  └─ 输出：采集的数据 + AI分析 + PM评论
  ↓
📊 深度情报
  └─ 用户痛点情报（VOC分析）
  └─ 技术资产盘点
  └─ 竞品功能矩阵
  └─ 输出：详细市场洞察 + 竞争格局
  ↓
🤝 人机共创
  └─ 阶段1：需求澄清（对话式）
  └─ 阶段2：红队压测（对话式）
  └─ 阶段3：用户替身访谈
  └─ 阶段4：专家评审
  └─ 输出：澄清后的需求 + 风险清单 + 用户反馈 + 专家意见
  ↓
🎯 人机协作量化决策
  └─ 五维评分（市场吸引力、用户需求强度、产品可行性、商业价值、风险等级）
  └─ PM调整权重和评论
  └─ AI融合分析
  └─ 输出：综合决策评分
  ↓
📐 方案对比
  └─ 规划（基础/期望/魅力属性）
  └─ 三套方案对比（高端/大众/性价比）
  └─ 文档输出
  └─ 输出：完整产品方案 + 可下载文档
```

## 4. 各页面详细设计

### 4.1 介绍页面
**功能**: 产品介绍和引导用户进入工作台  
**关键元素**:
- Hero区域: 主标题 + 副标题 + [开始体验]按钮
- 主要功能展示（4个核心功能）
- 技术亮点（多Agent系统、飞书集成）
- 工作流程可视化
- 页脚: 团队信息 + 联系方式

### 4.2 机会发现页面（全局）
**功能**: 展示AI自动发现的全品类机会排行榜  
**关键元素**:
- 评分维度权重说明
- 机会榜单（卡片形式）
  - 机会名称 + 综合评分
  - 机会描述
  - 关键发现（3-5条）
  - 分维度评分
  - [查看详情] [基于此机会创建项目]按钮

**Mock数据示例**:
```
🥇 全天办公/社交 AI 耳机 - 8.85分
   - 社交媒体：大量"通话不清"抱怨
   - 技术资产：HearID、AI降噪、741项专利
   - 竞争缺口：无人整合"清晰通话+翻译+透传"
```

### 4.3 新建项目页面
**功能**: 渐进式对话引导用户创建项目  
**Step 1 - 自由描述**:
- 大输入框（带提示文字，可删除）
- 支持拖拽上传文件
- 支持粘贴飞书文档链接
- [取消] [跳过，直接创建] [下一步]

**Step 2 - AI引导式问答**:
- 聊天记录区域（可滚动）
- 待回答问题列表（显示 X/5）
- 必答问题（❗标记）
- 可选问题（○标记）
- [一键跳过全部可选] [完成并创建项目]

**必答问题**:
1. 目标用户的主要使用场景
2. 决策紧急程度

**可选问题**:
1. 预期价格区间
2. 竞品参考
3. 市场背景
4. 技术背景

### 4.4 项目概览页面
**功能**: 展示项目基础信息和工作流进度  
**关键元素**:
- 项目基本信息卡片（名称、创建时间、状态）
- 工作流进度导航（可视化进度条）
  - 项目概览 → 信息看板 → 深度情报 → 人机共创 → 量化决策 → 方案对比
  - 显示当前进度和完成状态
- 快速跳转按钮

### 4.5 信息看板页面
**功能**: 数据采集 + AI初步分析 + 机会发现  
**关键元素**:

**部分1: 数据采集进度**
- 3个进度卡片（社交媒体、电商平台、行业报告）
- 显示已采集数量和进度百分比

**部分2: 各渠道AI初步分析**（可折叠卡片）
- 采集来源说明
- 关键发现（3-5条）
- AI建议
- PM评论区（可添加评论）
- AI融合分析（结合PM评论后的分析）

**部分3: 综合洞察**（可折叠卡片）
- 整合所有数据源的综合分析

**部分4: 机会发现**（基于当前项目数据）
- 基于采集数据发现的3个机会方向
- 每个机会包含：信号来源、市场空白、可行性

### 4.6 深度情报页面
**功能**: VOC分析 + 技术资产 + 竞品矩阵  
**关键元素**:

**用户痛点情报（表格）**:
| 痛点主题 | 真实信号 | 对产品的启示 |
|---------|---------|-------------|

**技术资产盘点（卡片展示）**:
- 专利数量
- 核心技术
- 用户基数
- TOP排名

**竞品功能矩阵（对比表格）**:
| 能力 | 我们的产品 | 竞品A | 竞品B | 竞品C |
|-----|----------|-------|-------|-------|

### 4.7 人机共创页面
**功能**: 需求澄清 + 红队压测 + 用户替身访谈 + 专家评审  
**关键元素**:

**进度条**: 显示当前阶段（1/4, 2/4, 3/4, 4/4）

**阶段1: 需求澄清**
- 对话界面
- AI提问，用户回答
- [跳过此阶段] [下一阶段]

**阶段2: 红队压测**
- 对话界面
- AI挑战，用户反驳
- [跳过此阶段] [下一阶段]

**阶段3: 用户替身访谈**
- 用户替身选择（4个卡片）
  - 小林（开放办公族）
  - Anna（跨国商务）
  - 阿杰（通勤学生）
  - 婷婷（敏感耳道）
- 对话界面（与选中的替身对话）
- [切换替身] [下一阶段]

**阶段4: 专家评审**
- 专家选择（3个选项）
  - 声学/算法专家
  - 人因工程专家
  - 供应链专家
- 显示专家评审结论
- [查看完整报告] [完成人机共创]

### 4.8 人机协作量化决策页面
**功能**: 五维评分 + PM调整 + 融合分析  
**关键元素**:

**五个维度卡片（默认折叠）**:
1. 📊 市场吸引力
2. 👥 用户需求强度
3. 🔧 产品可行性
4. 💰 商业价值
5. ⚠️ 风险等级

**每个卡片展开后包含**:
- AI评分 + PM判断输入框
- AI评分依据（4-5条，每条显示权重和得分）
- 权重调整滑块（可调整各因素权重）
- PM评论输入区
- AI融合分析（结合PM判断后的分析）

**综合决策建议**:
- 综合评分（满分5分）
- 决策建议（强烈推荐/推荐/有条件推荐/谨慎考虑/不推荐）

### 4.9 方案对比页面
**功能**: 规划 + 方案生成 + 文档输出  
**关键元素**:

**规划部分**（3列卡片）:
- 基础属性（Must）
- 期望属性（Performance）
- 魅力属性（Delight）

**方案对比部分**（对比表格）:
| 维度 | 方案A: 高端定位 | 方案B: 大众定位 | 方案C: 性价比定位 |
|-----|---------------|---------------|-----------------|
| 定位 | Premium | Mainstream | Value |
| 价格 | ¥999 | ¥599 | ¥399 |
| 核心功能 | ... | ... | ... |
| BOM成本 | ... | ... | ... |
| 毛利率 | ... | ... | ... |
| 开发周期 | ... | ... | ... |
| 风险评估 | ... | ... | ... |

**文档输出部分**:
- 产品提案汇总预览
- 决策审计日志
- [下载PDF] [下载PPT] [导出到飞书文档]

- 机会发现页面: `discovery.html`

### 设计原则
1. **一致性**: 所有页面使用统一的配色、字体、间距
2. **渐进式展示**: 信息默认折叠，避免信息过载
3. **人机协作**: PM可以在任何环节介入和调整
4. **可追溯性**: 所有决策和修改都有记录
